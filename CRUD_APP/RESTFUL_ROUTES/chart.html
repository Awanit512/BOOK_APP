RESTFUL Routes

A table of some important RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /books GET List all books present in our database book.find()
New /books/new GET Show new book form N/A
Create /books POST Create a new book, then redirect somewhere book.create()
Show /books/:id GET Show info about one specific book book.findById()
Edit /books/:id/edit GET Show edit form for one book book.findById()
Update /books/:id PUT Update particular book, then redirect somewhere book.findByIdAndUpdate()
Destroy /books/:id DELETE Delete a particular book, then redirect somewhere book.findByIdAndRemove()
